🏰 Description d’un Lieu Mystérieux

Décris une vieille maison abandonnée au sommet d'une colline par une nuit brumeuse.


✅ Grille d’auto-correction

  • Mon récit a un début, un milieu et une fin.
  • J’ai utilisé des descriptions et des dialogues pour rendre l’histoire vivante.

📜 Exemple de description

Sous une lune voilée, la vieille maison se dressait, ses fenêtres brisées laissant filtrer une lumière blafarde. Chaque pas sur le sol humide réveillait l’écho d’un passé oublié...

✍️ Conseils pour une bonne description

  • Utilise les cinq sens : décris ce que l’on voit, entend, sent, touche...
  • Ajoute des détails précis pour rendre le lieu plus vivant.
  • Crée une atmosphère en jouant avec la lumière, les ombres et les sons.
