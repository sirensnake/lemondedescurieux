🖥️ Les Bases de la Programmation

Bienvenue dans cette introduction à la programmation ! Apprends à donner des instructions à un ordinateur, comme si tu programmais un jeu vidéo.

🔢 Qu'est-ce qu'une instruction ?

Une instruction est un ordre donné à l'ordinateur. Par exemple :

avance(); // Curio avance d'un pas

Dans un jeu comme Minecraft, une instruction pourrait être "Avancer de 2 blocs".

📝 Les Variables

Une variable sert à stocker une valeur, comme une boîte contenant un objet.

nombreDeBlocs = 5;

Ici, nombreDeBlocs contient la valeur 5.

🔄 Les Boucles

Une boucle permet de répéter une action plusieurs fois sans la réécrire.

pour (i = 0; i < 5; i++) { avance(); // Curio avance 5 fois }

❓ Les Conditions

Les conditions permettent à un programme de prendre des décisions.

si (devantEstLibre()) { avance(); } sinon { tourneGauche(); }

Si Curio peut avancer, il avance ; sinon, il tourne à gauche.

🎮 Petit Exercice

Essaye de deviner ce que fera Curio :

pour (i = 0; i < 3; i++) { avance(); } tourneDroite();

Curio avance 3 fois puis tourne à droite.
