## CURIO THE FOX - DETAILED SPECIFICATIONS

**CRITICAL DESIGN REQUIREMENTS**:

### Visual Style
- **Pixel grid MUST be visible** - each pixel should be a clear square
- **Block-based construction** - fox made of visible cubic voxels
- **NO smooth curves** - all edges must be straight/stepped pixels
- **NO gradients** - use flat color blocks only

### Color Palette (Fox)
- **Primary**: Orange #ff6600 (body)
- **Secondary**: White #ffffff (chest/muzzle)
- **Accent**: Black #000000 (eyes, nose, outlines)
- **Optional**: Dark orange #cc5200 (shading, one shade only)

### Proportions
- **Head**: 32x32 pixel block (large and square)
- **Body**: 24x16 pixel block (rectangular)
- **Tail**: 16x24 pixel block (bushy, curved upward)
- **Legs**: 8x12 pixel blocks each (short and stubby)
- **Total height**: ~60-70 pixels

### Features
- **Eyes**: Two 3x3 pixel black squares, spaced apart
- **Nose**: Small black pixel cluster (triangular arrangement)
- **Ears**: Pointed triangular shapes made of pixels
- **Smile**: Optional curved line of pixels

### Texture Style
- Each color area should show a **visible pixel grid**
- Use **dithering** for any transitions (checkerboard pattern)
- **Outline** entire fox with 1-2px black border

### Pose & Expression
- **Sitting position**: Upright, friendly
- **Tail**: Curled upward behind body
- **Expression**: Friendly, curious (slight head tilt optional)
- **Style reference**: Exactly like Minecraft fox mob but simplified

### DO NOT
- ❌ Use smooth anti-aliasing
- ❌ Use gradient fills
- ❌ Make fox look realistic or smooth
- ❌ Add more than 2 shades per color
- ❌ Use rounded edges (everything must be pixelated steps)
