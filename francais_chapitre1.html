Chapitre 1 : Grammaire et Conjugaison

🔖 Les noms et les déterminants

Un nom est un mot qui sert à nommer une personne, un animal, un objet ou une idée. Les noms sont accompagnés de déterminants, qui les précèdent et donnent des informations comme le genre (masculin ou féminin) et le nombre (singulier ou pluriel).

Exemples :

  • Le chat : "le" est un déterminant masculin singulier.
  • Des pommes : "des" est un déterminant pluriel.

🌱 Les adjectifs qualificatifs

Un adjectif qualificatif est un mot qui apporte une précision ou une description à un nom. L’adjectif doit s’accorder avec le nom en genre et en nombre.

Exemple : Dans la phrase "Un chien noir", l’adjectif "noir" décrit le chien et s’accorde au masculin singulier.

⏳ Les temps de conjugaison

Les verbes peuvent se conjuguer à différents temps pour indiquer quand l’action a lieu. Voici les trois principaux temps de base :

  • Présent : L’action se passe maintenant. Exemple : "Je mange une pomme."
  • Futur : L’action se passera plus tard. Exemple : "Je mangerai une pomme."
  • Passé composé : L’action est déjà terminée. Exemple : "J’ai mangé une pomme."
