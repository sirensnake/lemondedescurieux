Les Supports d’Écriture à Travers le Temps 📜

Évolution des supports d’écriture

Le développement des supports d’écriture a accompagné les évolutions culturelles et technologiques des civilisations humaines.

1. Tablettes d’argile

Utilisées par les Mésopotamiens, ces tablettes étaient gravées à l’aide de roseaux.

2. Papyrus

Fabriqué à partir de la plante de papyrus, il a été largement utilisé par les Égyptiens.

3. Parchemin

Le parchemin, fait à partir de peaux animales, était plus résistant que le papyrus.

4. Papier

Inventé en Chine au IIe siècle avant J.-C., le papier est devenu le support d’écriture dominant dans le monde.

5. Supports modernes

Avec l’avènement des technologies numériques, les supports électroniques ont révolutionné l’écriture.
